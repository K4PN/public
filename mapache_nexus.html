Mapache Nexus · Versión móvil

Finalidad

Esta versión está pensada solo para generar y visualizar grafos de trazabilidad en móvil. No muestra investigaciones, marco izquierdo ni diario de gestiones.

Opciones incluidas

  • Conversor de Transferencias Bancarias: transforma columnas de origen, destino, titulares, fechas e importes en relaciones económicas.
  • Trazabilidad Blockchain: consulta exploradores compatibles y convierte movimientos públicos en relaciones económicas del grafo.
  • Grafo Canvas: permite mover nodos, desplazar el lienzo y hacer zoom con dos dedos.
  • Exportación PDF: genera una vista imprimible con el grafo y el resumen económico.

Uso rápido

Abre un conversor, genera el bloque y pulsa Añadir al grafo. El grafo se actualiza automáticamente. Puedes usar Centrar para recolocar la vista y Vaciar para empezar de nuevo.

Pegar trazabilidad para el grafo

Pega aquí una estructura ya preparada con el formato de Mapache Nexus. Se añadirá como una entrada interna oculta y el grafo se actualizará automáticamente.

Conversor de Transferencias Bancarias

Introduce un dato por línea. La línea 1 de cada bloque se relaciona con la línea 1 del resto. Los titulares son opcionales. Las cuentas se formatean como [Cuenta]. El importe se coloca antes de la cuenta destino: importe:[Cuenta destino]. Si añades fecha, se generará como fecha;importe:[Cuenta destino].

Trazabilidad Blockchain

Consulta movimientos públicos de una dirección y genera un bloque compatible con el grafo móvil de Mapache Nexus. Las API keys se guardan únicamente en el navegador mediante localStorage.
